# 影视排行榜

一款采用新拟态（Neumorphism)设计风格的微信小程序，用于浏览综艺、电影、热剧排行榜。

## 项目特点

- **新拟态设计**：柔和的渐变背景、软质凸起的光影效果、大圆角设计
- **子分类榜单**：每个主分类下有3个子分类，精细化内容展示
- **实时数据**：从TMDB/豆瓣API实时获取，无需手动导入
- **本地缓存**：30分钟缓存，下拉刷新清除缓存

## ⚠️ 使用说明

**本项目使用微信云开发个人免费版，存在以下限制：**

| 限制项 | 免费额度 | 影响 |
|--------|----------|------|
| 定时触发器 | ❌ 不支持 | 无法自动定时刷新榜单 |
| 云函数调用 | 1000次/天 | 高频使用可能超限 |
| 云函数超时 | 3秒 | 复杂操作可能超时 |
| 数据库读写 | 50000次/天 | 正常使用足够 |
| 存储空间 | 5GB | 足够使用 |

## 数据源

| 分类 | 子分类 | 数据源 | 说明 |
|------|--------|--------|------|
| 综艺 | 恋爱/搞笑/真人秀 | 豆瓣API | 只保留国内综艺 |
| 电影 | 热门/高分/最新 | TMDB API | 国际电影数据 |
| 热剧 | 韩剧/日剧 | TMDB API | 按语言/国家筛选 |
| 热剧 | 国产剧 | 豆瓣API | 国产电视剧 |

## 快速开始

### 1. 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取 AppID

### 2. 导入项目

1. 打开微信开发者工具
2. 导入项目目录
3. 填写 AppID

### 3. 开通云开发（个人免费版）

1. 点击工具栏「云开发」按钮
2. 选择「免费额度」开通
3. 记录环境 ID
4. 修改 `miniprogram/app.js` 中的环境 ID

### 4. 部署云函数

右键点击 `cloudfunctions/douban` 文件夹，选择「上传并部署：云端安装依赖」

### 5. 测试

1. 编译小程序
2. 点击首页右上角「调试」按钮
3. 点击「测试TMDB」和「测试豆瓣」检查连接状态
4. 如果TMDB超时，可能需要代理

## 分类结构

| 主分类 | 子分类 |
|--------|--------|
| 综艺 | 恋爱 / 搞笑 / 真人秀 |
| 电影 | 热门 / 高分 / 最新 |

每个子分类展示 **30部** 作品

## 设计风格

采用新拟态（Neumorphism）设计：

- **渐变背景**：左上浅天蓝 → 右下浅樱花粉
- **凸起效果**：左上高光阴影 + 右下深色阴影
- **凹陷效果**：激活状态使用内阴影
- **圆角设计**：大圆角，柔和视觉

## 抚摸技术栈

- 微信小程序原生框架
- 微信云开发（个人免费版）
- 云函数（Node.js）
- TMDB API（电影/剧集数据）
- 豆瓣API（综艺/国产剧数据）
