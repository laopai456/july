# 影视排行榜

一款采用新拟态（Neumorphism)设计风格的微信小程序，用于浏览综艺、电影、热剧排行榜。

## 项目特点

- **新拟态设计**：柔和的渐变背景、软质凸起的光影效果、大圆角设计
- **子分类榜单**：每个主分类下有3个子分类，精细化内容展示
- **实时数据**：从豆瓣API实时获取电影/热剧数据
- **本地缓存**：30分钟缓存，下拉刷新清除缓存

## 数据源

| 分类 | 子分类 | 数据源 | 说明 |
|------|--------|--------|------|
| 综艺 | 恋爱/搞笑/真人秀 | 本地数据 | 从豆瓣获取后存储在服务器 |
| 电影 | 热门/高分/最新 | 豆瓣API | 实时获取 |
| 热剧 | 韩剧/日剧/国产剧 | 豆瓣API | 实时获取 |

## 快速开始

### 1. 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取 AppID
- 云服务器（腾讯云等）

### 2. 导入项目

1. 打开微信开发者工具
2. 导入项目目录
3. 填写 AppID

### 3. 开通云开发

1. 点击工具栏「云开发」按钮
2. 开通云开发环境
3. 记录环境 ID
4. 修改 `miniprogram/app.js` 中的环境 ID

### 4. 部署云函数

右键点击 `cloudfunctions` 下的各文件夹，选择「上传并部署：云端安装依赖」

### 5. 部署后端服务

```bash
# 在云服务器上
cd /opt/movie-api
npm install express axios

# 创建 server.js（见项目根目录）
# 创建 data.json（综艺数据）

# 启动服务
node server.js

# 或使用 pm2 守护进程
pm2 start server.js --name movie-api
```

### 6. 更新综艺数据

```bash
# 运行数据获取脚本
node fetch_variety_simple.js
```

## 分类结构

| 主分类 | 子分类 |
|--------|--------|
| 综艺 | 恋爱 / 搞笑 / 真人秀 |
| 热剧 | 韩剧 / 日剧 / 国产剧 |
| 电影 | 热门 / 高分 / 最新 |

每个子分类展示 **30部** 作品

## 综艺分类规则

综艺根据标题关键词自动分类：

| 子分类 | 关键词示例 |
|--------|-----------|
| 恋爱 | 心动的信号、女儿们的恋爱、再见爱人、非诚勿扰、桃花、妻子... |
| 搞笑 | 脱口秀、吐槽大会、王牌对王牌、奔跑吧、极限挑战... |
| 真人秀 | 其他所有综艺（默认分类） |

## 设计风格

采用新拟态（Neumorphism）设计：

- **渐变背景**：左上浅天蓝 → 右下浅樱花粉
- **凸起效果**：左上高光阴影 + 右下深色阴影
- **凹陷效果**：激活状态使用内阴影
- **圆角设计**：大圆角，柔和视觉

## 技术栈

- 微信小程序原生框架
- 微信云开发
- 云函数（Node.js）
- Express.js（后端服务）
- 豆瓣API（数据源）
